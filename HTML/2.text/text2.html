Block Element
포함된 내용만으로 한 라인을 독점
영역 구조를 만들 때 사용
레이아웃 구성
Inline Element
포함된 내용과 앞 뒤 태그의 내용을 같은 라인에 출력
문서를 구성하는 역할
블록요소나 다른 인라인 요소 안에서 쓰인다
Void Element
시작태그와 끝 태그 사이에 아무것도 포함되지 않은 요소
비어있는 요소가 항상 인라인 요소는 아님
자기폐쇄형 요소
콘텐츠를 가지지 않으므로 비어있지 않은 요소가 될 수 없다
Non-void Element
문자열이나 인라인 요소를 담을 수 있는 요소(끝 태그가 존재)
콘텐츠를 가지지 않아도 닫는 태그를 넣어줘야 함
다른 블록태그를 포함할 수 있음
